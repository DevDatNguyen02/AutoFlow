## Data Flow

1. **Client Interaction:** Users interact with the Next.js frontend.
2. **API Requests:** Frontend makes HTTP requests to the NestJS backend.
3. **Business Logic:** NestJS controllers hand off to services for processing.
4. **Database Access:** Services use Prisma Client to interact with the PostgreSQL database.
5. **Response:** Data is returned via REST API to the frontend.
